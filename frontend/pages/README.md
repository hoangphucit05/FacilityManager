# Pages

Thư mục này chứa các trang HTML theo nhóm tính năng:

- `auth/`: màn hình đăng nhập/xác thực
- `dashboard/`: quản lý tài sản, thống kê, danh mục, tòa nhà
- `profile/`: quản lý user, liên hệ, hồ sơ
- `settings/`: cài đặt hệ thống

Trang hiện tại đã có:

- `dashboard/`: `assets.html`, `categories.html`, `departments.html`, `liquidation.html`, `statistics.html`
- `profile/`: `users.html`, `users-add.html`, `contact.html`, `contact-profile.html`

Tai lieu ket noi backend:

- `API_CONTRACT.md`: mapping field `name` frontend sang DTO backend va endpoint de xuat.
