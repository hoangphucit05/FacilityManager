# Study Package

Package gốc của backend.

## Thư mục con

- `controller/`: REST API
- `service/`: nghiệp vụ
- `repository/`: truy cập dữ liệu
- `entity/`: mô hình DB
- `dto/`: dữ liệu request/response
- `mapper/`: chuyển đổi entity/dto
- `config/`: cấu hình
- `exception/`: xử lý lỗi
- `common/`: thành phần dùng chung
- `anotation/`: custom annotation (nếu dùng)
