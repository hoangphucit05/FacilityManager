# Controller

## Cần làm

- Tạo các REST controller theo contract trong `frontend/pages/API_CONTRACT.md`.
- Chuẩn hóa endpoint theo prefix `/api`.
- Validate request (Bean Validation) và trả về response rõ ràng.
- Không viết nghiệp vụ trong controller, chỉ điều phối sang service.
- Bổ sung swagger annotation cho endpoint quan trọng.
