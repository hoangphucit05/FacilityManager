# Exception

## Cần làm

- Tạo `GlobalExceptionHandler` với `@RestControllerAdvice`.
- Chuẩn hóa format lỗi trả về frontend (code, message, details, timestamp).
- Mapping lỗi validation, business, not found, conflict.
- Log lỗi server mức độ phù hợp, tránh lộ thông tin nhạy cảm.
- Viết test cho các case lỗi quan trọng.
