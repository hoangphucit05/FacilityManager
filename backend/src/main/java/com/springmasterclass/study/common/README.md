# Common

## Cần làm

- Đặt class dùng chung: constants, enums, util, response wrapper.
- Tránh để logic nghiệp vụ domain trong thư mục này.
- Chuẩn hóa convention đặt tên để dễ tìm.
- Tách riêng helper cho date/time, paging, text normalize nếu cần.
- Bổ sung unit test cho util có tính toán.
